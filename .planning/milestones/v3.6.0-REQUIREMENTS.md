## Phase 31 — Idempotent Execution Round-Trip (Exactly-Once-Effect)

Source: `31-SPEC.md` (ambiguity 0.13). VERIFICATION: **passed 8/8**; SECURITY: 19/19 threats closed (`threats_open: 0`); VALIDATION: verified, `nyquist_compliant: true`, 8/8 COVERED (hermetic 445/0).

- [x] **req-1 — Deterministic identity (H)**: every dispatch/result carries `H = SHA-256(correlationId, workflowId, stepId, processorId, EntryId)` with `executionId` EXCLUDED; a retry/redelivery/re-dispatch of the same logical message yields the same `H`. — **Satisfied** (golden tests pin determinism + executionId-independence).
- [x] **req-2 — Per-fire identity + entry-step EntryId**: `correlationId` is the per-fire id; entry steps carry a deterministic non-empty `EntryId = hash(correlationId, stepId)`; the "skip input read" signal moved to `InputDefinition == null`. — **Satisfied**.
- [x] **req-3 — Content-addressed two-level L2 data**: result blobs at `data[hash(result)]`; the manifest `[hash(r1)…]` at `data[EntryId]` where `EntryId = hash(manifest)`; empty result → `manifest = []` → terminal branch; schema validation on the blob, not the manifest. — **Satisfied**.
- [x] **req-4 — Effect-first CAS dedup (both hops)**: `flag[H] = Pending|Ack` dedup at processor (inbound `EntryStepDispatch`) AND orchestrator (inbound `ExecutionResult`); the effect is produced BEFORE the atomic `Pending → Ack` CAS, so a crash yields a *collapsed duplicate*, never a lost branch. — **Satisfied**.
- [x] **req-5 — Merge correctness (per-edge via input EntryId, content-collapse)**: per-edge executions distinguished by input `EntryId` (different output → distinct `H`, no override); identical-input branches collapse; per-edge, not a join; `predecessorStepId` NOT in `H`. — **Satisfied**.
- [x] **req-6 — Manifest fan-out**: the orchestrator unbundles the manifest and dispatches one successor per (item × `NextStep`) carrying the successor's `stepId`/`processorId`, the item `EntryId`, and a regenerated `executionId` (N×M dispatches, each with its own deterministic `H`). — **Satisfied**.
- [x] **req-7 — Configurable retry + L2 key format**: retry budget bound from appsettings (default `Immediate(N)`); content-addressed `data`/`flag` keys are `prefix + 64-hex` (golden-pinned). — **Satisfied**.
- [x] **req-8 — Live exactly-once-effect proof (real-stack)**: the merge topology no longer duplicates downstream — `IdempotentExactlyOnceE2ETests` (merge topology across fires + induced `Immediate(N)`/redelivery) shows the expected per-fire effect set with zero extra execution (the `StepB4`-×2 inverse). — **Satisfied** (live-proven inside the 31.1 + 32.1 close gates; redis net-zero held).
