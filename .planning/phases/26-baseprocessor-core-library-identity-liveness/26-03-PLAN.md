| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-26-10 | Denial of Service | Redis bounce crashing the host | mitigate | D-11 log-and-continue: each `StringSetAsync` is wrapped in try/catch that logs a warning and continues the loop (never throws/returns); the soft-dep multiplexer (`abortConnect=false`) materializes even when Redis is down. Proven by LivenessResilienceFacts. |
