## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → ASP.NET Core MVC pipeline | Untrusted HTTP route segment (`{sourceHash}`) crosses here. Bound via ASP.NET model binding into a `string` parameter. |
| Service layer → EF Core / Npgsql | Untrusted string is forwarded into a LINQ predicate `p => p.SourceHash == sourceHash` translated to a parameterized SQL query by EF Core 8 + Npgsql 8.0.9 (no string concatenation). |

|-----------|----------|-----------|-------------|-----------------|

WHERE source_hash = $1`). Verified by reading Phase 3's BaseDbContext usage — no raw SQL, no `FromSqlRaw`, no string concatenation in this method. |
