```csharp
private const char Sep = '\u001F';   // ASCII unit separator — never appears in Guid "D" or lowercase hex (D-03 reserved separator, Tampering mitigation T-31-01)

private static string Hex(string canonicalText)
{
    using var sha = System.Security.Cryptography.SHA256.Create();
    var bytes = sha.ComputeHash(System.Text.Encoding.UTF8.GetBytes(canonicalText));
    var sb = new System.Text.StringBuilder(64);
    foreach (var b in bytes) sb.Append(b.ToString("x2"));   // LOWERCASE x2, never X2 / Convert.ToHexString
    return sb.ToString();
}
```

Public surface (each Guid rendered `g.ToString("D")` — invariant, lowercase by default in .NET; EntryId passed as its existing 64-hex string; FIXED field order):
```csharp
// req-1: the dedup identity. executionId is DELIBERATELY absent (D-02 — lineage only, excluded from H).
public static string ComputeH(Guid correlationId, Guid workflowId, Guid stepId, Guid processorId, string entryId)
    => Hex(string.Join(Sep, correlationId.ToString("D"), workflowId.ToString("D"), stepId.ToString("D"), processorId.ToString("D"), entryId));

// req-3: content-address a result blob. Same core path.
public static string HashBlob(string blob) => Hex(blob);

// req-3 / D-08: content-address the manifest. Pass the ALREADY-serialized JSON array text (caller owns JsonSerializer.Serialize(string[])) so the hash is over the exact wire bytes.
public static string HashManifest(string manifestJson) => Hex(manifestJson);

// req-2: deterministic entry-step EntryId = hash(correlationId, stepId).
public static string EntryEntryId(Guid correlationId, Guid stepId)
    => Hex(string.Join(Sep, correlationId.ToString("D"), stepId.ToString("D")));
```
Do NOT add a JSON dependency here — `HashManifest` takes the already-serialized string so this leaf stays serialization-agnostic and the manifest bytes are owned by the caller (the processor) consistently. Use only `System.Security.Cryptography` + `System.Text`.
